## 📝 前言

AI处理的是**不确定性**。概率论是描述不确定性的数学语言。

本章将用直观的方式讲解概率论基础。

## 🔰 1. 概率基础

### 1.1 什么是概率？

**概率 = 事件发生的可能性**

```
抛硬币：P(正面) = 0.5
掷骰子：P(6) = 1/6
```

### 1.2 概率的性质

```
0 ≤ P(A) ≤ 1
P(必然事件) = 1
P(不可能事件) = 0
P(A) + P(¬A) = 1
```

### 1.3 生活例子

| 事件 | 概率 |
|------|------|
| 明天下雨 | 0.3 |
| 中彩票 | 0.000001 |
| 明天太阳升起 | ≈1 |

## 🔰 2. 条件概率

### 2.1 什么是条件概率？

**在A发生的条件下，B发生的概率**

```
P(B|A) = P(A∩B) / P(A)
```

### 2.2 生活例子

```
已知：今天下雨（A）
问：今天带伞的概率（B）

P(带伞|下雨) = P(下雨且带伞) / P(下雨)
```

### 2.3 代码示例

```python
# 条件概率
def conditional_prob(p_ab, p_a):
    """P(B|A) = P(A∩B) / P(A)"""
    return p_ab / p_a

# 例子：已知下雨概率0.3，下雨且带伞概率0.2
p_rain = 0.3
p_rain_and_umbrella = 0.2

p_umbrella_given_rain = conditional_prob(p_rain_and_umbrella, p_rain)
print(f"下雨时带伞的概率：{p_umbrella_given_rain:.2f}")  # 0.67
```

## 🔰 3. 贝叶斯定理

### 3.1 公式

```
P(A|B) = P(B|A) * P(A) / P(B)
```

### 3.2 直观理解

```
后验概率 = 似然 × 先验 / 证据

P(疾病|检测阳性) = P(阳性|疾病) × P(疾病) / P(阳性)
```

### 3.3 生活例子

```
疾病检测：
- 疾病发病率：P(疾病) = 0.01
- 检测准确率：P(阳性|疾病) = 0.99
- 假阳性率：P(阳性|无病) = 0.05

问题：检测阳性，真的有病的概率？

P(疾病|阳性) = 0.99 × 0.01 / (0.99×0.01 + 0.05×0.99)
             = 0.0099 / 0.0594
             ≈ 0.167

结论：只有16.7%的概率真的有病！
```

### 3.4 代码示例

```python
def bayes_theorem(p_b_given_a, p_a, p_b):
    """贝叶斯定理"""
    return (p_b_given_a * p_a) / p_b

# 疾病检测例子
p_disease = 0.01  # 先验：疾病发病率
p_positive_given_disease = 0.99  # 似然：有病时检测阳性
p_positive = 0.99 * 0.01 + 0.05 * 0.99  # 证据：检测阳性概率

p_disease_given_positive = bayes_theorem(
    p_positive_given_disease,
    p_disease,
    p_positive
)

print(f"检测阳性时真正有病的概率：{p_disease_given_positive:.2%}")
```

## 🔰 4. 常见分布

### 4.1 离散分布

#### 伯努利分布

```
只有两种结果：成功(1)或失败(0)

P(X=1) = p
P(X=0) = 1-p
```

#### 二项分布

```
n次独立伯努利试验中成功的次数

P(X=k) = C(n,k) * p^k * (1-p)^(n-k)
```

### 4.2 连续分布

#### 正态分布（高斯分布）

```
最重要的分布！

f(x) = (1/√(2πσ²)) * exp(-(x-μ)²/(2σ²))

μ: 均值
σ: 标准差
```

#### 均匀分布

```
所有值等概率

f(x) = 1/(b-a), a ≤ x ≤ b
```

### 4.3 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 正态分布
mu, sigma = 0, 1
x = np.linspace(-4, 4, 100)
y = stats.norm.pdf(x, mu, sigma)

plt.plot(x, y)
plt.title("正态分布 (μ=0, σ=1)")
plt.xlabel("x")
plt.ylabel("概率密度")
plt.show()

# 生成随机样本
samples = np.random.normal(mu, sigma, 1000)
print(f"样本均值：{samples.mean():.2f}")
print(f"样本标准差：{samples.std():.2f}")
```

## 🔰 5. 最大似然估计

### 5.1 核心思想

**选择让数据出现概率最大的参数**

### 5.2 生活例子

```
抛硬币10次，7次正面，3次反面
问：P(正面)是多少？

最大似然估计：P(正面) = 7/10 = 0.7
```

### 5.3 数学公式

```
L(θ) = ∏ P(x_i | θ)

取对数：
log L(θ) = ∑ log P(x_i | θ)

求导等于0，得到最优参数
```

## 🔰 6. AI中的应用

### 6.1 朴素贝叶斯分类器

```python
from sklearn.naive_bayes import GaussianNB
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_iris

# 加载数据
iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3
)

# 训练
model = GaussianNB()
model.fit(X_train, y_train)

# 预测
accuracy = model.score(X_test, y_test)
print(f"准确率：{accuracy:.2%}")
```
