# 📈 数学基础：微积分与优化

> **理解梯度下降的数学原理**

---

## 📝 前言

深度学习的核心是**梯度下降**，而梯度下降的基础是**微积分**。

本章将直观讲解微积分的核心概念。

---

## 🔰 1. 导数

### 1.1 什么是导数？

**导数 = 函数在某点的变化率**

```
f'(x) = lim(Δx→0) [f(x+Δx) - f(x)] / Δx
```

### 1.2 直观理解

```
想象你在山上：
- 导数告诉你：往哪个方向走，海拔变化最快
- 正导数：上坡
- 负导数：下坡
- 零导数：平地（山顶或山谷）
```

### 1.3 常见导数

```
d/dx (x^n) = nx^(n-1)
d/dx (e^x) = e^x
d/dx (ln(x)) = 1/x
d/dx (sin(x)) = cos(x)
```

---

## 🔰 2. 偏导数

### 2.1 什么是偏导数？

**多变量函数对某一个变量的导数**

```
f(x, y) = x² + y²

∂f/∂x = 2x  （把y当常数）
∂f/∂y = 2y  （把x当常数）
```

### 2.2 直观理解

```
想象一个曲面：
- 偏导数告诉你：沿x方向或y方向的斜率
- 就像站在山坡上，分别看东西方向和南北方向的坡度
```

---

## 🔰 3. 梯度

### 3.1 什么是梯度？

**梯度 = 所有偏导数组成的向量**

```
∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z, ...)
```

### 3.2 梯度的意义

```
梯度方向：函数值增长最快的方向
负梯度方向：函数值下降最快的方向
```

### 3.3 代码示例

```python
import numpy as np

def f(x, y):
    """目标函数"""
    return x**2 + y**2

def gradient(x, y):
    """计算梯度"""
    df_dx = 2 * x
    df_dy = 2 * y
    return np.array([df_dx, df_dy])

# 测试
x, y = 3, 4
grad = gradient(x, y)
print(f"点({x},{y})的梯度：{grad}")
print(f"梯度方向：{grad / np.linalg.norm(grad)}")
```

## 🔰 4. 链式法则

### 4.1 什么是链式法则？

**复合函数的导数 = 外层导数 × 内层导数**

```
d/dx [f(g(x))] = f'(g(x)) × g'(x)
```

### 4.2 生活例子

```
y = (2x + 1)³

令 u = 2x + 1
则 y = u³

dy/dx = dy/du × du/dx
       = 3u² × 2
       = 6(2x + 1)²
```

### 4.3 为什么重要？

**反向传播就是链式法则的应用！**

```
神经网络：x → h1 → h2 → y

∂y/∂x = ∂y/∂h2 × ∂h2/∂h1 × ∂h1/∂x
```

## 🔰 5. 梯度下降

### 5.1 核心思想

**沿着负梯度方向更新参数，直到最小值**

```
θ_new = θ_old - α × ∇f(θ_old)

α: 学习率
∇f: 梯度
```

### 5.2 生活比喻

```
蒙眼下山：
1. 感觉脚下坡度（计算梯度）
2. 往最陡的下坡方向走一步（更新参数）
3. 重复直到平地（收敛）
```

### 5.3 代码示例

```python
import numpy as np

def gradient_descent(learning_rate=0.1, n_steps=100):
    """梯度下降求最小值"""
    # 初始点
    x = 5.0
    history = [x]
    
    for _ in range(n_steps):
        # 计算梯度
        grad = 2 * x  # f(x) = x² 的导数
        
        # 更新
        x = x - learning_rate * grad
        history.append(x)
    
    return x, history

# 运行
minimum, history = gradient_descent()
print(f"最小值点：{minimum:.4f}")
print(f"函数最小值：{minimum**2:.4f}")
```

## 🔰 6. 优化算法

### 6.1 梯度下降变体

| 算法 | 特点 |
|------|------|
| SGD | 随机梯度下降 |
| Momentum | 动量加速 |
| AdaGrad | 自适应学习率 |
| Adam | 结合Momentum和AdaGrad |

### 6.2 Adam算法

```python
class Adam:
    def __init__(self, lr=0.001, beta1=0.9, beta2=0.999, eps=1e-8):
        self.lr = lr
        self.beta1 = beta1
        self.beta2 = beta2
        self.eps = eps
        self.m = 0
        self.v = 0
        self.t = 0
    
    def update(self, param, grad):
        self.t += 1
        self.m = self.beta1 * self.m + (1 - self.beta1) * grad
        self.v = self.beta2 * self.v + (1 - self.beta2) * grad**2
        
        m_hat = self.m / (1 - self.beta1**self.t)
        v_hat = self.v / (1 - self.beta2**self.t)
        
        param = param - self.lr * m_hat / (np.sqrt(v_hat) + self.eps)
        return param
```
