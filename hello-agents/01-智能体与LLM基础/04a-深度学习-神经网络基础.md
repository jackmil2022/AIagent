## 🔰 1. 从生物神经元到人工神经元

### 1.1 生物神经元

```
树突（输入）→ 细胞体（处理）→ 轴突（输出）

多个输入 → 加权求和 → 激活 → 输出
```

### 1.2 人工神经元

```
输入：x1, x2, ..., xn
权重：w1, w2, ..., wn
偏置：b

输出 = f(Σ(wi * xi) + b)
```

### 1.3 代码实现

```python
import numpy as np

def perceptron(inputs, weights, bias):
    """单个感知机"""
    # 加权求和
    weighted_sum = np.dot(inputs, weights) + bias
    
    # 激活函数
    output = 1 if weighted_sum > 0 else 0
    
    return output

# 示例
inputs = np.array([1, 0, 1])
weights = np.array([0.5, -0.3, 0.8])
bias = -0.2

result = perceptron(inputs, weights, bias)
print(f"输出：{result}")
```

## 🔰 2. 激活函数

### 2.1 为什么需要激活函数？

**没有激活函数，神经网络只是线性变换！**

```
无激活：y = W2(W1x + b1) + b2 = W'x + b'（还是线性）
有激活：y = f(W2(f(W1x + b1)) + b2)（非线性）
```

### 2.2 常见激活函数

#### Sigmoid

```python
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# 输出范围：(0, 1)
# 用途：二分类输出层
```

#### ReLU

```python
def relu(x):
    return max(0, x)

# 输出范围：[0, ∞)
# 用途：隐藏层（最常用）
```

#### Tanh

```python
def tanh(x):
    return (np.exp(x) - np.exp(-x)) / (np.exp(x) + np.exp(-x))

# 输出范围：(-1, 1)
# 用途：RNN
```

### 2.3 对比

| 函数 | 范围 | 优点 | 缺点 |
|------|------|------|------|
| Sigmoid | (0,1) | 输出概率 | 梯度消失 |
| ReLU | [0,∞) | 计算简单 | 死神经元 |
| Tanh | (-1,1) | 零中心 | 梯度消失 |

## 🔰 3. 前向传播

### 3.1 计算过程

```
输入层 → 隐藏层1 → 隐藏层2 → 输出层

每一层：
1. 线性变换：z = Wx + b
2. 激活：a = f(z)
```

### 3.2 代码实现

```python
import numpy as np

class NeuralNetwork:
    def __init__(self, layers):
        self.layers = layers
        self.weights = []
        self.biases = []
        
        # 初始化权重
        for i in range(len(layers) - 1):
            w = np.random.randn(layers[i], layers[i+1]) * 0.01
            b = np.zeros((1, layers[i+1]))
            self.weights.append(w)
            self.biases.append(b)
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def forward(self, x):
        """前向传播"""
        a = x
        
        for i in range(len(self.weights)):
            z = np.dot(a, self.weights[i]) + self.biases[i]
            a = self.relu(z)
        
        return a

# 创建网络
nn = NeuralNetwork([3, 5, 2])
print("网络结构：3 → 5 → 2")

# 前向传播
x = np.random.randn(1, 3)
output = nn.forward(x)
print(f"输入形状：{x.shape}")
print(f"输出形状：{output.shape}")
```

## 🔰 4. 损失函数

### 4.1 什么是损失函数？

**衡量模型预测与真实值的差距**

### 4.2 常见损失函数

#### 均方误差（MSE）

```
MSE = Σ(y - ŷ)² / n

用于回归任务
```

#### 交叉熵损失

```
CrossEntropy = -Σ y * log(ŷ)

用于分类任务
```

### 4.3 代码实现

```python
def mse_loss(y_true, y_pred):
    """均方误差"""
    return np.mean((y_true - y_pred) ** 2)

def cross_entropy_loss(y_true, y_pred):
    """交叉熵损失"""
    return -np.mean(y_true * np.log(y_pred + 1e-8))

# 示例
y_true = np.array([1, 0, 1])
y_pred = np.array([0.9, 0.1, 0.8])

print(f"MSE：{mse_loss(y_true, y_pred):.4f}")
print(f"交叉熵：{cross_entropy_loss(y_true, y_pred):.4f}")
```

## 🔰 5. 反向传播

### 5.1 核心思想

**计算每个参数对损失的贡献（梯度）**

### 5.2 链式法则

```
∂L/∂w = ∂L/∂a × ∂a/∂z × ∂z/∂w
```

### 5.3 直观理解

```
想象多米诺骨牌：
- 最后一块倒了（损失）
- 问：哪块骨牌贡献最大？（梯度）
- 方法：从后往前，逐个计算影响
```
