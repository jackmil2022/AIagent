# 🤖 机器学习：基本概念

> **让计算机从数据中学习**

---

## 📝 前言

机器学习是AI的核心。本章将带你理解机器学习的基本概念。

---

## 🔰 1. 什么是机器学习？

### 1.1 定义

**机器学习 = 让计算机从数据中自动学习规律**

### 1.2 生活例子

| 任务 | 传统编程 | 机器学习 |
|------|----------|----------|
| 垃圾邮件过滤 | 写规则 | 从邮件数据学习 |
| 人脸识别 | 写特征 | 从图片数据学习 |
| 推荐系统 | 写规则 | 从用户行为学习 |

### 1.3 为什么需要ML？

```
有些任务太复杂，无法写明确的规则：
- 识别猫的图片
- 理解自然语言
- 预测股票走势
```

## 🔰 2. 机器学习类型

### 2.1 监督学习

**有标签的学习**

```
输入(X) → 模型 → 预测(y) → 与标签对比 → 调整
```

#### 分类问题

```
输入：邮件内容
输出：垃圾/非垃圾
```

#### 回归问题

```
输入：房屋特征
输出：房价
```

### 2.2 无监督学习

**没有标签的学习**

#### 聚类

```
输入：用户数据
输出：用户分组
```

#### 降维

```
输入：高维数据
输出：低维表示
```

### 2.3 强化学习

**通过奖励学习**

```
Agent → 动作 → 环境 → 奖励 → Agent
```

## 🔰 3. 核心概念

### 3.1 数据集划分

| 划分 | 用途 | 比例 |
|------|------|------|
| 训练集 | 训练模型 | 70% |
| 验证集 | 调参 | 15% |
| 测试集 | 最终评估 | 15% |

### 3.2 过拟合与欠拟合

```
欠拟合：模型太简单，学不到规律
过拟合：模型太复杂，记住了噪声
```

#### 生活比喻

```
考试：
- 欠拟合：没复习，不会做
- 刚好：理解知识，会举一反三
- 过拟合：死记硬背，换个题就不会
```

### 3.3 评估指标

#### 分类指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 准确率 | (TP+TN)/(TP+TN+FP+FN) | 整体正确率 |
| 精确率 | TP/(TP+FP) | 预测为正的准确率 |
| 召回率 | TP/(TP+FN) | 正样本被找出的比例 |
| F1 | 2*P*R/(P+R) | 精确率和召回率的调和 |

#### 回归指标

| 指标 | 公式 | 说明 |
|------|------|------|
| MSE | Σ(y-ŷ)²/n | 均方误差 |
| MAE | Σ|y-ŷ|/n | 平均绝对误差 |
| R² | 1 - SS_res/SS_tot | 决定系数 |

## 🔰 4. 代码示例

### 4.1 scikit-learn入门

```python
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score
from sklearn.datasets import load_iris

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# 划分数据
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 训练模型
model = LogisticRegression(max_iter=200)
model.fit(X_train, y_train)

# 预测
y_pred = model.predict(X_test)

# 评估
accuracy = accuracy_score(y_test, y_pred)
print(f"准确率：{accuracy:.2%}")
```
