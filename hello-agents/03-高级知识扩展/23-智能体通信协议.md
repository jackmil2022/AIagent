# 🔗 智能体通信协议

> **让AI Agent能够相互通信和协作**

---

## 📝 前言

单个Agent能力有限，多个Agent协作可以完成更复杂的任务。

但Agent之间如何通信？这就需要**通信协议**。

---

## 🔰 1. 为什么需要通信协议？

### 1.1 生活类比

想象一个公司：
- 销售、研发、财务需要协作
- 但他们说不同的"语言"
- 需要统一的沟通协议（如邮件、会议）

### 1.2 Agent通信的挑战

| 挑战 | 说明 |
|------|------|
| 异构性 | 不同框架、不同语言 |
| 可发现性 | 如何找到其他Agent |
| 互操作性 | 如何理解彼此的消息 |

## 🔰 2. MCP 协议

### 2.1 什么是MCP？

**MCP = Model Context Protocol（模型上下文协议）**

由Anthropic提出，用于连接AI模型和外部工具/数据。

### 2.2 核心概念

| 概念 | 说明 |
|------|------|
| Host | 宿主应用（如IDE） |
| Client | MCP客户端 |
| Server | MCP服务器（提供工具） |

### 2.3 工作流程

```
Host → Client → Server
              ↓
         执行工具
              ↓
         返回结果
```

### 2.4 代码示例

```python
# MCP Server
from mcp import Server, Tool

server = Server("my-server")

@server.tool()
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city}今天晴，25°C"

@server.tool()
def search_web(query: str) -> str:
    """搜索网页"""
    # 实现搜索逻辑
    return f"搜索结果：{query}"

# 启动服务器
server.run()
```

### 2.5 客户端调用

```python
from mcp import Client

# 连接服务器
client = Client("http://localhost:8000")

# 列出可用工具
tools = client.list_tools()
print("可用工具：", tools)

# 调用工具
result = client.call_tool("get_weather", {"city": "北京"})
print("结果：", result)
```

## 🔰 3. A2A 协议

### 3.1 什么是A2A？

**A2A = Agent-to-Agent Protocol**

由Google提出，用于Agent之间的直接通信。

### 3.2 核心特性

| 特性 | 说明 |
|------|------|
| Agent发现 | 发现其他Agent |
| 任务委托 | 委托任务给其他Agent |
| 状态同步 | 同步任务状态 |

### 3.3 Agent描述

```json
{
  "name": "weather-agent",
  "description": "提供天气查询服务",
  "url": "http://localhost:8001",
  "capabilities": ["weather_query", "forecast"],
  "authentication": {
    "type": "bearer",
    "token": "xxx"
  }
}
```

### 3.4 任务委托

```python
# Agent A 委托任务给 Agent B
task = {
    "id": "task-001",
    "type": "weather_query",
    "input": {"city": "北京"},
    "callback_url": "http://agent-a:8000/callback"
}

# 发送任务
response = post("http://agent-b:8001/tasks", json=task)

# 接收结果
@app.route("/callback", methods=["POST"])
def callback():
    result = request.json
    print(f"任务完成：{result}")
```

## 🔰 4. ANP 协议

### 4.1 什么是ANP？

**ANP = Agent Network Protocol**

专注于大规模Agent网络的协议。

### 4.2 核心特性

| 特性 | 说明 |
|------|------|
| 去中心化 | 无中心服务器 |
| 可扩展性 | 支持大量Agent |
| 安全性 | 内置安全机制 |

### 4.3 网络架构

```
Agent ←→ Agent ←→ Agent
  ↕          ↕          ↕
Agent ←→ Agent ←→ Agent
```

---

## 🔰 5. 协议对比

### 5.1 详细对比

| 特性 | MCP | A2A | ANP |
|------|-----|-----|-----|
| 提出者 | Anthropic | Google | 开源社区 |
| 用途 | 工具连接 | Agent通信 | 大规模网络 |
| 复杂度 | 低 | 中 | 高 |
| 成熟度 | 高 | 中 | 低 |

### 5.2 选择建议

| 场景 | 推荐协议 |
|------|----------|
| 连接工具和数据 | MCP |
| 两个Agent协作 | A2A |
| 大规模Agent网络 | ANP |

## 🔰 6. 实战示例

### 6.1 多Agent协作系统

```python
# Agent定义
class WeatherAgent:
    def __init__(self):
        self.tools = ["get_weather", "get_forecast"]
    
    def handle_task(self, task):
        if task["type"] == "weather_query":
            city = task["input"]["city"]
            return self.get_weather(city)

class TranslatorAgent:
    def __init__(self):
        self.tools = ["translate"]
    
    def handle_task(self, task):
        if task["type"] == "translate":
            text = task["input"]["text"]
            target_lang = task["input"]["target"]
            return self.translate(text, target_lang)

# 协调器
class AgentCoordinator:
    def __init__(self):
        self.agents = {
            "weather": WeatherAgent(),
            "translator": TranslatorAgent()
        }
    
    def route_task(self, task):
        agent_type = task.get("agent_type")
        if agent_type in self.agents:
            return self.agents[agent_type].handle_task(task)
        return "未知任务类型"

# 使用
coordinator = AgentCoordinator()

# 查询天气
weather_task = {
    "agent_type": "weather",
    "type": "weather_query",
    "input": {"city": "北京"}
}
result = coordinator.route_task(weather_task)
print(result)  # 北京今天晴，25°C
```

### 6.2 使用LangGraph实现

```python
from langgraph.graph import StateGraph, END
from langchain_openai import ChatOpenAI

# 定义状态
class AgentState:
    task: str
    result: str = None
    next_agent: str = None

# 定义Agent节点
def weather_agent(state):
    """天气Agent"""
    # 处理天气查询
    return {"result": "北京今天晴"}

def translator_agent(state):
    """翻译Agent"""
    # 处理翻译任务
    return {"result": "Beijing is sunny today"}

# 构建图
workflow = StateGraph(AgentState)
workflow.add_node("weather", weather_agent)
workflow.add_node("translator", translator_agent)

# 定义边
workflow.add_conditional_edges(
    "weather",
    lambda state: "translator" if state.get("translate") else END
)

# 编译
app = workflow.compile()
```
