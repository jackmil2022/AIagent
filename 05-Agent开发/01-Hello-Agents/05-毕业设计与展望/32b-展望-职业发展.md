## 🔰 2. 学习路径

### 2.1 初学者路线（0-6个月）

```
第1-2月：Python基础
    ↓
第3-4月：机器学习入门
    ↓
第5-6月：深度学习基础
```

**学习内容**：
- Python语法、NumPy、Pandas
- 线性回归、决策树、SVM
- 神经网络、CNN、RNN

**推荐资源**：
- 吴恩达《机器学习》
- 李沐《动手学深度学习》

### 2.2 进阶路线（6-12个月）

```
第7-8月：NLP基础
    ↓
第9-10月：Transformer与预训练
    ↓
第11-12月：LLM应用开发
```

**学习内容**：
- 词向量、序列模型
- Transformer、BERT、GPT
- RAG、Agent开发

**推荐资源**：
- HuggingFace课程
- LangChain文档

### 2.3 高级路线（12个月+）

```
深入研究方向：
├── 模型训练与微调
├── 多模态模型
├── Agent系统设计
└── MLOps与部署
```

## 🔰 4. 面试准备

### 4.1 常见问题

#### 基础知识

| 问题 | 要点 |
|------|------|
| Transformer原理 | 自注意力、多头注意力、位置编码 |
| BERT vs GPT | 双向vs单向、预训练任务 |
| RAG原理 | 检索增强、向量数据库 |
| LoRA原理 | 低秩适配、参数高效 |

#### 工程实践

| 问题 | 要点 |
|------|------|
| 如何优化RAG？ | 分块策略、重排序、混合检索 |
| 如何评估LLM？ | 基准测试、人工评估 |
| 如何部署模型？ | 量化、vLLM、TensorRT |

#### 系统设计

| 问题 | 要点 |
|------|------|
| 设计智能客服 | 意图识别、知识库、对话管理 |
| 设计代码助手 | 代码理解、生成、测试 |
| 设计RAG系统 | 架构、检索、生成 |

### 4.2 面试技巧

1. **STAR法则**：Situation-Task-Action-Result
2. **项目深挖**：能说清楚每个技术选型的理由
3. **动手能力**：现场写代码、调bug
4. **学习能力**：展示快速学习新技术的能力
