## 1. 巨量引擎（抖音）投流Agent教程

### 1.1 平台概述

| 项目 | 说明 |
|------|------|
| 官方网址 | [oceanengine.com](https://www.oceanengine.com) |
| 覆盖平台 | 抖音、今日头条、西瓜视频、番茄小说等 |
| 广告形式 | 信息流广告、开屏广告、搜索广告、直播间引流等 |
| 计费模式 | CPM/CPC/oCPM/oCPC/CPA/ROI出价 |

### 1.2 投流Agent基本操作流程

#### 步骤1：注册与开户
1. 访问巨量引擎官网，点击"立即注册"
2. 选择账户类型（企业账户/个人账户）
3. 完成企业资质认证（营业执照、行业资质等）
4. 绑定广告主账户并完成充值

#### 步骤2：创建广告计划
1. 进入广告管理后台，点击"新建计划"
2. 选择营销目标：
   - 品牌曝光
   - 线索收集（表单、电话、在线咨询）
   - 应用推广（APP下载）
   - 电商转化（商品购买、直播间引流）
3. 设置投放预算（日预算/总预算）
4. 选择投放时段和地域

#### 步骤3：定向设置
| 定向类型 | 说明 | 操作要点 |
|---------|------|---------|
| 基础定向 | 年龄、性别、地域 | 初期可适当放宽 |
| 兴趣标签 | 用户兴趣偏好 | 选择与产品相关的3-5个兴趣标签 |
| 行为定向 | 用户搜索/浏览行为 | 结合兴趣标签使用 |
| DMP人群包 | 自定义人群 | 上传CRM数据或使用lookalike扩展 |
| 设备定向 | 手机品牌、操作系统 | 根据产品特性设置 |

#### 步骤4：出价策略
| 出价方式 | 说明 | 适用场景 |
|---------|------|---------|
| CPM | 按千次曝光付费 | 品牌曝光、追求展示量 |
| CPC | 按点击付费 | 追求点击量 |
| oCPC | 按转化优化的点击出价 | 转化目标明确 |
| oCPM | 按转化优化的千次曝光出价 | **最常用，适合效果广告** |
| CPA | 按实际转化付费 | 严格控制成本 |
| ROI出价 | 按目标ROI出价 | 电商、直播带货 |

#### 步骤5：创意制作
- **视频素材**：竖版9:16格式，时长15-60秒
- **图片素材**：高清、有吸引力、突出产品卖点
- **文案撰写**：简洁有力，突出利益点，包含行动号召

#### 步骤6：提交审核与上线
- 素材提交后系统审核（通常1-2个工作日）
- 审核通过后自动开始投放
- 实时监控数据表现

### 1.3 oCPM出价核心技巧

#### 出价设置技巧
| 技巧 | 具体操作 |
|------|---------|
| 初始出价 | 从行业参考出价的1.2-1.5倍开始 |
| 阶梯出价法 | 创建多条计划，设置高/中/低三档出价 |
| 冷启动期 | 适当提高出价10%-20%，帮助系统快速学习 |
| 放量期 | 逐步降低出价，每次调整幅度不超过5%-10% |

#### 预算设置技巧
| 技巧    | 具体操作                     |
| ----- | ------------------------ |
| 计划预算  | 设置为出价的20-30倍（至少能转化20次以上） |
| 调整幅度  | 每次调整不超过20%               |
| 测试期策略 | 小预算多计划测试                 |

#### 学习期注意事项
| 注意事项 | 说明 |
|---------|------|
| 学习期转化量 | 通常需要20-50个转化才能完成学习 |
| 修改限制 | 学习期内不要频繁修改出价、预算、定向 |
| 无转化处理 | 3天无转化建议直接新建计划 |

### 1.4 高级功能与技巧

#### DMP人群包使用
1. **第一方人群包**：上传自有数据（手机号、设备号等）
2. **人群扩展**：基于种子用户进行lookalike扩展
3. **应用事件人群**：基于用户App内行为事件筛选
4. **电商人群**：基于电商行为（购买、加购等）圈选

#### 自动化投放工具
| 工具    | 功能              |
| ----- | --------------- |
| 智能投放  | AI自动优化出价和人群定向   |
| 素材生成  | AI辅助生成广告素材      |
| 自动化规则 | 设置条件自动暂停/开启计划   |
| 数据洞察  | AI分析投放数据，给出优化建议 |

### 1.5 常见问题与解决方案

| 问题    | 原因分析          | 优化建议              |
| ----- | ------------- | ----------------- |
| 跑不出去量 | 出价过低/人群过窄/素材差 | 提高出价、放宽定向、更换素材    |
| 成本过高  | 出价过高/人群不精准    | 降低出价、优化定向、排除无效人群  |
| 成本波动大 | 转化数据不足/频次过高   | 增加转化量、扩大投放范围      |
| 前期成本高 | 学习期正常波动       | 耐心等待，至少积累20+转化再调整 |
| 后期不起量 | 素材衰退/人群覆盖饱和   | 更新素材、扩展定向         |

### 1.7 成功案例分析

#### 案例：某教育品牌抖音投放
- **背景**：K12在线教育课程推广
- **目标**：获取表单线索
- **策略**：
  - 使用oCPM出价，目标成本50元
  - 人群定向：25-45岁，一二线城市，教育兴趣标签
  - 素材：真人出镜+痛点文案
- **结果**：
  - 日均获取线索200+
  - 单条线索成本35-45元
  - ROI达到1:3

### 1.8 数据分析与优化方法

#### 关键指标监控
| 指标 | 说明 | 优化方向 |
|------|------|---------|
| 曝光量 | 广告展示次数 | 调整出价、扩展定向 |
| 点击率（CTR） | 点击量/曝光量 | 优化素材、标题 |
| 转化率（CVR） | 转化量/点击量 | 优化落地页、表单设计 |
| 单次转化成本 | 总花费/转化量 | 调整出价、优化定向 |
| ROI | 转化价值/广告花费 | 综合优化所有环节 |

#### 优化节奏
| 时间 | 操作 |
|------|------|
| 投放前2小时 | 密切监控曝光和点击情况 |
| 投放4-6小时 | 分析初步数据，判断是否需要调整 |
| 投放1-3天 | 根据数据表现进行优化调整 |
| 每周 | 复盘整体数据，调整投放策略 |
