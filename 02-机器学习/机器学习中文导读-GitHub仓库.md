## 7. 英文看不懂怎么办

只抓 4 类信息：

1. `README`：看项目是干什么的；
2. 目录名：判断章节顺序；
3. Notebook 代码：先运行，不急着逐句翻译；
4. 图片和输出：理解模型效果。

遇到英文概念，先查中文关键词：

| 英文 | 中文 |
|---|---|
| regression | 回归 |
| classification | 分类 |
| clustering | 聚类 |
| feature | 特征 |
| label / target | 标签 / 目标 |
| loss | 损失 |
| overfitting | 过拟合 |
| underfitting | 欠拟合 |
| validation | 验证 |
| evaluation | 评估 |
| metric | 指标 |
| pipeline | 流水线 |
