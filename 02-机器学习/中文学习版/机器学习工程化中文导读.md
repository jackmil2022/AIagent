# 机器学习工程化中文导读

对应源文件：[[../源文件/GitHub原文/Made-With-ML/README|Made-With-ML]]

## 1. 工程化解决什么

Notebook 里模型跑通，不等于项目能上线。工程化关注：

- 数据是否稳定；
- 训练是否可复现；
- 模型是否可测试；
- 服务是否可部署；
- 上线后是否能监控；
- 模型变差时是否能发现。

## 2. 学习前提

先满足：

- 会训练至少一个 sklearn 模型；
- 会保存模型；
- 会写项目说明；
- 做过至少 2 个端到端项目。

## 3. 最小工程化清单

- [ ] 固定随机种子；
- [ ] 保存训练参数；
- [ ] 保存模型文件；
- [ ] 保存评估指标；
- [ ] 写数据检查；
- [ ] 写预测函数；
- [ ] 写最小 API 或命令行入口；
- [ ] 记录模型版本。
