# 微软机器学习入门课程中文导读

对应源文件：[[../源文件/GitHub原文/ML-For-Beginners/README|ML-For-Beginners]]

## 1. 课程定位

这是一门适合初学者的经典机器学习课程，覆盖：

- 机器学习历史和基本概念；
- 回归；
- 分类；
- 聚类；
- 自然语言处理；
- 时间序列；
- 强化学习；
- 真实世界应用。

## 2. 中文学习路线

| 原课程模块 | 中文学习目标 | 先学程度 |
|---|---|---|
| Introduction | 知道机器学习是什么 | 必学 |
| Regression | 会预测连续数值 | 必学 |
| Classification | 会预测类别 | 必学 |
| Clustering | 会无监督分组 | 必学 |
| NLP | 知道文本怎么变成特征 | 选学 |
| Time Series | 知道时间序列预测 | 选学 |
| Reinforcement | 知道强化学习是什么 | 了解 |
| Real World | 看真实应用 | 了解 |

## 3. 推荐学习顺序

1. 先读本项目 [[../机器学习零基础中文速学]]；
2. 打开源文件课程的 Introduction；
3. 跑 Regression 里的 Notebook；
4. 跑 Classification 里的 Notebook；
5. 用自己的话写总结；
6. 再做 [[../../91-项目实战/01-房价预测/房价预测项目说明|房价预测项目]]。

## 4. 重点概念中文对照

| 英文 | 中文 |
|---|---|
| dataset | 数据集 |
| feature | 特征 |
| label / target | 标签 / 目标 |
| regression | 回归 |
| classification | 分类 |
| clustering | 聚类 |
| model | 模型 |
| training | 训练 |
| evaluation | 评估 |
| metric | 指标 |
