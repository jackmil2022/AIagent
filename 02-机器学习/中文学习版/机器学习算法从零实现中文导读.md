## 1. 什么时候看从零实现

不要一开始就看。从零实现适合在你已经会用 sklearn 之后，用来理解算法内部。

## 2. 推荐顺序

1. 线性回归；
2. Logistic 回归；
3. KNN；
4. 决策树；
5. 随机森林；
6. K-Means；
7. PCA；
8. SVM。

## 3. 看代码时只抓四件事

| 问题 | 例子 |
|---|---|
| 输入是什么 | `X`, `y` |
| 参数是什么 | `weights`, `bias` |
| 损失是什么 | MSE、交叉熵 |
| 怎么更新 | 梯度下降、分裂节点、迭代聚类 |

## 4. 第一个手写算法

先手写线性回归：

1. 初始化 `w` 和 `b`；
2. 预测 `y_hat = X @ w + b`；
3. 计算 MSE；
4. 计算梯度；
5. 更新参数；
6. 画 loss 曲线。
