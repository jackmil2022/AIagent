## 使用指南

### 新手入门
1. 从`01-优化师基础知识-定义分类与核心技能.md`开始阅读
2. 重点学习前5章内容（定义、技能、流程、数据、创意）
3. 结合`02-优化师实用工具与资源.md`中的工具学习
4. 参考`03-优化师实战案例与最佳实践.md`中的案例实践

### 进阶提升
1. 深入学习`01-优化师基础知识-定义分类与核心技能.md`中的高级内容
2. 掌握`02-优化师实用工具与资源.md`中的专业工具
3. 研究`03-优化师实战案例与最佳实践.md`中的最佳实践
4. 学习进阶技巧和趋势应对策略

### 职业发展
1. 参考职业发展路径章节
2. 制定个人学习计划
3. 建立个人品牌
4. 持续学习和提升
