## 1. 命令行参数

### 1.1 argparse 使用

`argparse` 是 Python **标准库**中的命令行参数解析模块，用于在运行脚本时从终端接收参数。核心作用：将命令行输入的字符串自动解析为 Python 变量，避免每次修改参数都要改代码。

```python
import argparse

# 1. 创建解析器
parser = argparse.ArgumentParser(description='训练脚本')

# 2. 添加参数
parser.add_argument('--dataroot', required=True, help='数据集路径')
#                              ↑ 必须传，不传报错

parser.add_argument('--name', default='experiment', help='实验名称')
#                             ↑ 不传就用默认值

parser.add_argument('--model', default='cycle_gan', help='模型类型')

parser.add_argument('--lr', type=float, default=0.001, help='学习率')
#                          ↑ 自动转为 float 类型

parser.add_argument('--use_gpu', action='store_true', help='使用GPU')
#                               ↑ 传了就是 True，不传就是 False

# 3. 解析
args = parser.parse_args()

# 4. 使用
print(args.dataroot)   # './datasets/maps'
print(args.lr)         # 0.001
print(args.use_gpu)    # True 或 False
```

**常用参数类型：**

| 参数 | 说明 |
|------|------|
| `required=True` | 必填参数 |
| `default=值` | 可选，有默认值 |
| `type=int/float/str` | 自动类型转换 |
| `action='store_true'` | 布尔开关（传了为True） |
| `nargs='+'` | 接收多个值（如 `--files a.txt b.txt`） |

**为什么开源项目都用它？**
1. **灵活性** — 不用改代码就能切换数据集、模型、超参数
2. **可复用** — 一个脚本适用于多种实验配置
3. **自文档化** — `python train.py --help` 自动显示所有参数说明

### 1.2 命令行运行

```bash
# 基本运行
python train.py --dataroot ./datasets/maps --name maps_cyclegan

# 添加参数
python train.py --dataroot ./datasets/maps \
                --name maps_cyclegan \
                --model cycle_gan \
                --lr 0.0002 \
                --use_gpu
```

## 2. 查看开源项目

### 2.1 项目结构

```
project/
├── README.md          # 项目说明
├── requirements.txt   # 依赖
├── train.py           # 训练脚本
├── test.py            # 测试脚本
├── models/            # 模型定义
├── datasets/          # 数据加载
├── options/           # 配置选项
└── utils/             # 工具函数
```

### 2.2 阅读顺序

1. **README.md**：了解项目功能
2. **requirements.txt**：安装依赖
3. **train.py**：理解训练流程
4. **models/**：查看模型结构
5. **datasets/**：理解数据加载

## 3. 实用技巧

### 3.1 使用 TensorBoard

```python
from torch.utils.tensorboard import SummaryWriter

writer = SummaryWriter('runs/experiment_1')

# 记录标量
writer.add_scalar('loss/train', loss.item(), epoch)

# 记录图像
writer.add_images('images', images, epoch)

writer.close()
```

### 3.2 模型可视化

```python
# 使用 torchsummary
from torchsummary import summary

model = resnet18()
summary(model, input_size=(3, 224, 224))
```
