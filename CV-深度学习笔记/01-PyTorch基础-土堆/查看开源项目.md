## 1. 命令行参数

### 1.1 argparse 使用

```python
import argparse

parser = argparse.ArgumentParser(description='训练脚本')

# 必需参数
parser.add_argument('--dataroot', required=True, help='数据集路径')

# 可选参数
parser.add_argument('--name', default='experiment', help='实验名称')
parser.add_argument('--model', default='cycle_gan', help='模型类型')
parser.add_argument('--lr', type=float, default=0.001, help='学习率')

# 布尔参数
parser.add_argument('--use_gpu', action='store_true', help='使用GPU')

args = parser.parse_args()
```

### 1.2 命令行运行

```bash
# 基本运行
python train.py --dataroot ./datasets/maps --name maps_cyclegan

# 添加参数
python train.py --dataroot ./datasets/maps \
                --name maps_cyclegan \
                --model cycle_gan \
                --lr 0.0002 \
                --use_gpu
```

## 2. 查看开源项目

### 2.1 项目结构

```
project/
├── README.md          # 项目说明
├── requirements.txt   # 依赖
├── train.py           # 训练脚本
├── test.py            # 测试脚本
├── models/            # 模型定义
├── datasets/          # 数据加载
├── options/           # 配置选项
└── utils/             # 工具函数
```

### 2.2 阅读顺序

1. **README.md**：了解项目功能
2. **requirements.txt**：安装依赖
3. **train.py**：理解训练流程
4. **models/**：查看模型结构
5. **datasets/**：理解数据加载

## 3. 实用技巧

### 3.1 使用 TensorBoard

```python
from torch.utils.tensorboard import SummaryWriter

writer = SummaryWriter('runs/experiment_1')

# 记录标量
writer.add_scalar('loss/train', loss.item(), epoch)

# 记录图像
writer.add_images('images', images, epoch)

writer.close()
```

### 3.2 模型可视化

```python
# 使用 torchsummary
from torchsummary import summary

model = resnet18()
summary(model, input_size=(3, 224, 224))
```
