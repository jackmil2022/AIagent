# 转置卷积是一种卷积

> 来源：李沐《动手学深度学习》

---

## 1. 什么是转置卷积？

### 1.1 普通卷积 vs 转置卷积

| 普通卷积 | 转置卷积 |
|----------|----------|
| 输入大 → 输出小 | 输入小 → 输出大 |
| 下采样 | 上采样 |
| 提取特征 | 恢复尺寸 |

### 1.2 直观理解

```
普通卷积 (2x2 → 1x1):
[1 2]     [1 2 3 4] * kernel = [output]
[3 4]

转置卷积 (1x1 → 2x2):
[output] → [? ?]
            [? ?]
```

---

## 2. 为什么叫"转置"？

### 2.1 数学原理

普通卷积可以表示为矩阵乘法：

```
y = C * x

转置卷积使用 C 的转置：
y = C^T * x
```

### 2.2 直观解释

```
普通卷积：4x4 输入 → 2x2 输出
[1 2 3 4]    [a b c d]   [ ]
[5 6 7 8]  * [e f g h] = [ ]
[9 10 11 12] [i j k l]   
[13 14 15 16][m n o p]   

转置卷积：2x2 输入 → 4x4 输出
使用相同的权重，但"反向"操作
```

## 3. PyTorch 实现

```python
import torch
import torch.nn as nn

# 转置卷积层
conv_transpose = nn.ConvTranspose2d(
    in_channels=3,
    out_channels=16,
    kernel_size=4,
    stride=2,
    padding=1
)

# 输入
x = torch.randn(1, 3, 32, 32)

# 输出
output = conv_transpose(x)
print(f"输入: {x.shape}")    # (1, 3, 32, 32)
print(f"输出: {output.shape}")  # (1, 16, 64, 64)
```

## 5. 注意事项

### 5.1 棋盘效应

转置卷积可能产生**棋盘格伪影**：

```
解决方案：
1. 使用 Resize + Conv 代替 ConvTranspose
2. 调整 kernel_size 和 stride
3. 添加后处理层
```

### 5.2 替代方案

```python
# 方法1：上采样 + 卷积
upsample = nn.Upsample(scale_factor=2, mode='bilinear')
conv = nn.Conv2d(3, 16, 3, padding=1)

# 方法2：像素洗牌 (PixelShuffle)
pixel_shuffle = nn.PixelShuffle(upscale_factor=2)
```
