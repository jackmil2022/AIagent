## 2. 实战技能

### 2.1 模型训练

```python
# 标准训练流程
1. 数据准备 → DataLoader
2. 模型定义 → nn.Module
3. 损失函数 → nn.CrossEntropyLoss
4. 优化器 → optim.Adam
5. 训练循环 → forward + backward + step
6. 评估 → 验证集指标
```

### 2.2 调参技巧

| 技巧 | 说明 |
|------|------|
| 学习率 | 从 1e-3 开始，用余弦退火 |
| 批大小 | 32/64/128 |
| 正则化 | Dropout(0.1-0.5), Weight Decay(1e-4) |
| 数据增强 | 随机裁剪、翻转、颜色抖动 |

## 3. 进阶学习路径

### 3.1 CV 方向

```
基础 CNN
  ↓
目标检测 (YOLO, Faster R-CNN)
  ↓
语义分割 (U-Net, DeepLab)
  ↓
实例分割 (Mask R-CNN)
  ↓
视觉 Transformer (ViT, Swin)
```

### 3.2 NLP 方向

```
RNN/LSTM
  ↓
Transformer
  ↓
预训练模型 (BERT, GPT)
  ↓
大语言模型 (LLaMA, ChatGPT)
  ↓
AI Agent
```
