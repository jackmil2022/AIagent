# 优化算法

> 来源：李沐《动手学深度学习》

---

## 1. 优化问题

### 1.1 核心目标

**找到使损失函数最小的参数**

```
min L(θ)
```

### 1.2 挑战

- **局部最小值**：可能陷入非全局最优
- **鞍点**：梯度为零但不是最值
- **病态条件**：不同方向曲率差异大

---

## 2. 梯度下降变体

### 2.1 批量梯度下降 (BGD)

```python
# 使用全部数据计算梯度
for epoch in range(num_epochs):
    gradient = compute_gradient(model, entire_dataset)
    model.parameters -= lr * gradient
```

### 2.2 随机梯度下降 (SGD)

```python
# 每次只用一个样本
for epoch in range(num_epochs):
    for x, y in dataset:
        gradient = compute_gradient(model, x, y)
        model.parameters -= lr * gradient
```

### 2.3 小批量 SGD (Mini-batch SGD)

```python
# 每次用一个小批量
for epoch in range(num_epochs):
    for batch in dataloader:
        gradient = compute_gradient(model, batch)
        model.parameters -= lr * gradient
```

---

## 3. 动量法 (Momentum)

### 3.1 核心思想

**累积历史梯度，加速收敛**

```python
v = β * v + gradient  # β 通常取 0.9
parameters -= lr * v
```

### 3.2 直观理解

```
想象一个球滚下山坡：
- 沿坡方向加速
- 横跨坡的振荡被抑制
```

---

## 4. AdaGrad

### 4.1 核心思想

**自适应学习率，频繁更新的参数用小学习率**

```python
cache += gradient ** 2
parameters -= lr * gradient / (sqrt(cache) + ε)
```

### 4.2 优缺点

| 优点 | 缺点 |
|------|------|
| 自适应学习率 | 学习率单调递减 |
| 适合稀疏数据 | 后期学习太慢 |

---

## 5. RMSProp

### 5.1 核心思想

**用指数移动平均改进 AdaGrad**

```python
cache = β * cache + (1 - β) * gradient ** 2
parameters -= lr * gradient / (sqrt(cache) + ε)
```

---

## 6. Adam

### 6.1 核心思想

**结合 Momentum 和 RMSProp**

```python
# 一阶矩估计（均值）
m = β1 * m + (1 - β1) * gradient

# 二阶矩估计（方差）
v = β2 * v + (1 - β2) * gradient ** 2

# 偏差修正
m_hat = m / (1 - β1 ** t)
v_hat = v / (1 - β2 ** t)

# 更新参数
parameters -= lr * m_hat / (sqrt(v_hat) + ε)
```

### 6.2 超参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| lr | 0.001 | 学习率 |
| β1 | 0.9 | 一阶矩衰减率 |
| β2 | 0.999 | 二阶矩衰减率 |
| ε | 1e-8 | 数值稳定项 |

## 7. PyTorch 使用

```python
import torch.optim as optim

# SGD
optimizer = optim.SGD(model.parameters(), lr=0.01, momentum=0.9)

# Adam
optimizer = optim.Adam(model.parameters(), lr=0.001)

# AdamW (推荐)
optimizer = optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)

# 学习率调度器
scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=100)
```

---

## 8. 对比总结

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| SGD | 简单 | 收敛慢 | 基准 |
| Momentum | 加速收敛 | 需调参 | 通用 |
| AdaGrad | 自适应 | 学习率衰减 | 稀疏数据 |
| RMSProp | 改进 AdaGrad | - | RNN |
| Adam | 综合最优 | 可能泛化差 | 通用首选 |
