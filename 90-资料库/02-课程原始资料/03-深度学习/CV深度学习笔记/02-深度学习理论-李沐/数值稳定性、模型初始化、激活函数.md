# 数值稳定性、模型初始化、激活函数

# 1. 数值稳定性




① MLP：多层感知机。

② 对角矩阵(diagonal matrix)是一个主对角线之外的元素皆为0的矩阵，常写为diag(a1, a2, ..., an)。

③ diag * W 把diag和W分开看。这就是个链式求导，diag是n维度的relu向量對n维度relu的输入的求导，向量对自身求导就是对角矩阵。

# 3. 梯度消失

① 蓝色为原函数，黄色为梯度函数。


② 当激活函数的输入稍微大一点时，它的导数就变为接近0，连续n个接近0的数相乘，最后的梯度就接近0，梯度就消失了。



# 4. 总结

① **数值稳定性**：深度网络训练中，梯度可能爆炸或消失，导致训练不稳定或无法收敛。

② **梯度爆炸**：权重矩阵的元素值过大时，梯度在反向传播中连乘导致指数级增长。

③ **梯度消失**：激活函数（如 Sigmoid）在输入较大时导数趋近于 0，梯度在反向传播中逐渐衰减。

④ 解决方案：
- 合理的权重初始化（如 Xavier、He 初始化）
- 使用 ReLU 等不会饱和的激活函数
- 批量归一化（Batch Normalization）
- 残差连接（Residual Connection）

⑤ ReLU 是目前最常用的激活函数：$f(x) = \max(0, x)$，计算简单且缓解梯度消失。
