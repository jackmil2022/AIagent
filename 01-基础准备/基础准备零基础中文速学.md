# 基础准备零基础中文速学

这篇只讲进入机器学习之前最重要的基础。你不需要先成为 Python 专家，也不需要先学完高等数学。

## 1. 你到底要准备什么

机器学习需要四种基础：

| 基础 | 作用 | 不会会怎样 |
|---|---|---|
| Python | 写代码、调用库、处理文件 | 跑不动教程 |
| 数据处理 | 清洗表格、看数据分布 | 不知道模型吃了什么 |
| 数学直觉 | 理解模型为什么能训练 | 公式看不懂 |
| 工具习惯 | Notebook、虚拟环境、GitHub | 项目容易乱 |

## 2. Python 最小知识

你先掌握这些：

```python
# 变量
price = 100
name = "house"

# 列表
features = ["area", "rooms", "age"]

# 字典
sample = {"area": 80, "rooms": 3, "price": 200}

# 条件
if sample["price"] > 150:
    print("expensive")

# 循环
for feature in features:
    print(feature)

# 函数
def price_per_area(price, area):
    return price / area
```

能看懂这段，就能开始 pandas。

## 3. pandas 最小知识

pandas 是处理表格数据的工具。机器学习里最常见的数据就是表格。

```python
import pandas as pd

df = pd.read_csv("house.csv")

print(df.shape)        # 行数、列数
print(df.head())       # 前 5 行
print(df.info())       # 字段类型
print(df.describe())   # 数值统计
print(df.isna().sum()) # 缺失值
```

你要习惯问：

1. 数据有多少行？
2. 有多少列？
3. 哪一列是标签？
4. 哪些列是特征？
5. 有没有缺失值？
6. 有没有明显异常值？

## 4. 数学最小直觉

### 向量

一条样本可以看成一个向量：

$$
x = [80, 3, 10]
$$

含义：面积 80、房间 3、房龄 10。

### 矩阵

很多样本放在一起就是矩阵：

$$
X =
\begin{bmatrix}
80 & 3 & 10 \\
60 & 2 & 5 \\
120 & 4 & 20
\end{bmatrix}
$$

机器学习模型通常吃的就是这个 `X`。

### 均值和方差

- 均值：平均水平；
- 方差：波动大小。

如果房价波动很大，模型更难预测。

### 导数和梯度

导数表示“往哪个方向变化”。梯度告诉模型参数该往哪里改，才能让错误变小。

## 5. 第一次数据探索

用任何表格数据做：

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("data.csv")

print(df.head())
print(df.isna().sum())

df.hist(figsize=(10, 8))
plt.show()
```

然后写中文结论：

- 哪些字段最重要？
- 哪些字段缺失多？
- 哪些字段分布奇怪？
- 标签大概长什么样？
- 下一步需要清洗什么？
