## 2. 只学够用的 Python

| 必会内容 | 为什么要会 | 自测 |
|---|---|---|
| 变量和类型 | 数据都是对象，表格字段也有类型 | 能解释 `int`、`float`、`str`、`list`、`dict` |
| 条件判断 | 数据清洗经常要按条件处理 | 能写 `if/elif/else` |
| 循环 | 批量处理文件或字段 | 能遍历列表和字典 |
| 函数 | 把重复逻辑封装起来 | 能写带参数和返回值的函数 |
| 文件读写 | 读 CSV、保存结果 | 能读写文本文件 |
| 异常处理 | 处理文件不存在、字段缺失 | 能写 `try/except` |

先不要纠结装饰器、元类、异步编程。机器学习入门用不到。

## 3. 数学只补四件事

| 数学模块 | 先理解什么 | 机器学习里怎么用 |
|---|---|---|
| 线性代数 | 向量、矩阵、点积、矩阵乘法 | 特征矩阵、线性模型、神经网络 |
| 概率统计 | 均值、方差、分布、条件概率 | 数据分布、模型评估、朴素贝叶斯 |
| 微积分 | 导数、偏导、梯度 | 梯度下降、反向传播 |
| 优化 | 损失函数、学习率、局部最优 | 训练模型 |

判断是否够用：你能看懂下面这句话就够进入机器学习：

> 模型预测值和真实值之间有误差，损失函数衡量误差，优化算法根据梯度一点点调整参数，让损失变小。

## 4. 数据处理最小能力

拿到一个 CSV，你至少要会：

1. 看行数列数；
2. 看每列类型；
3. 看缺失值；
4. 看数值列分布；
5. 看类别列取值；
6. 找异常值；
7. 画图；
8. 写中文结论。

常用代码概念：

```python
import pandas as pd

df = pd.read_csv("data.csv")
df.head()
df.info()
df.describe()
df.isna().sum()
df["列名"].value_counts()
```

## 5. 入门项目：数据体检报告

选一个表格数据集，写一份中文数据体检报告。

报告结构：

1. 数据是什么；
2. 每行代表什么；
3. 每列代表什么；
4. 哪些列有缺失；
5. 哪些列可能是标签；
6. 哪些特征可能有用；
7. 画 3 张图；
8. 写 5 条观察；
9. 写 3 个可能的数据问题；
10. 写下一步建模计划。
