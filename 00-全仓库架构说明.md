# 全仓库架构说明

## 顶层职责

| 目录 | 职责 |
|---|---|
| `00-*` | 全局入口、总路线、总索引、治理说明 |
| `01-基础准备` | Python、数学、数据处理基础 |
| `02-机器学习` | 机器学习主线学习内容 |
| `03-深度学习` | 深度学习主线学习内容 |
| `04-大模型与NLP` | Transformer、LLM、NLP 主线内容 |
| `05-Agent开发` | Agent、RAG、工作流与工具调用 |
| `06-行业应用` | 面向业务场景的 Agent 应用 |
| `90-资料库` | 原文镜像、课程原始资料、研究报告、素材 |
| `91-项目实战` | 端到端项目 |
| `99-归档` | 历史版本和已迁移旧内容 |

## 判定规则

- 直接学习的内容留在 `01-06`、`91`
- 参考原件和镜像放进 `90-资料库`
- 历史版本和旧目录放进 `99-归档`
