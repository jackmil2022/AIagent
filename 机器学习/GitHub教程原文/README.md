## 使用顺序

1. 先看 [[../机器学习从零到高手学习路径|机器学习从零到高手学习路径]]。
2. 零基础先读“微软机器学习入门课程”，练手用 `Kaggle Learn` 在线课程。
3. 想做 Notebook 实战，看“从零到精通机器学习实战课”和“动手学机器学习配套代码”。
4. 想理解算法内部，看“机器学习算法从零实现”和“常见机器学习算法手写实现”。
5. 想进入深度学习，看“动手学深度学习中文版”、“神经网络从零到英雄”、“PyTorch 官方教程”。
6. 想工程化，看“机器学习工程化实战”。
