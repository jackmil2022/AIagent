## 外部资源索引

- [Microsoft ML-For-Beginners](https://github.com/microsoft/ML-For-Beginners)：完整入门课程，适合先建立大图景。
- [Daniel Bourke: Zero to Mastery Machine Learning](https://github.com/mrdbourke/zero-to-mastery-ml)：Notebook 风格，适合跟着敲。
- [Aurélien Géron: Hands-On Machine Learning notebooks](https://github.com/ageron/handson-ml3)：经典实战书配套 Notebook。
- [Kaggle Learn: Intro to Machine Learning](https://www.kaggle.com/learn/intro-to-machine-learning)：短练习，适合快速获得正反馈。
- [DeepLearning.AI Machine Learning Specialization](https://www.deeplearning.ai/courses/machine-learning-specialization/)：系统课，适合补数学和算法直觉。
- [ML-From-Scratch](https://github.com/eriklindernoren/ML-From-Scratch)：适合学完基础后看算法内部。
- [Made With ML](https://github.com/GokuMohandas/Made-With-ML)：适合学完基础后看工程化。
